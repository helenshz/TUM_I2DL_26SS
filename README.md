# TUM_I2DL_IN2346_26SS

This repository contains the programming exercises and learning notes for the **Introduction to Deep Learning (I2DL)** course at the Technical University of Munich (TUM).

## Course Overview
* **Course Name**: Introduction to Deep Learning (I2DL)
* **University**: Technical University of Munich (TUM)
* **Instructor**: Prof. Matthias Niessner
* **Objective**: Gain a deep understanding of the fundamental building blocks of deep learning through theoretical lectures and hands-on programming experiments (Interactive Coding Notebooks), master common deep learning libraries, and build a solid foundation for independent deep learning research projects.

## Teaching Team
* **Lecturer**: Prof. Matthias Niessner
* **Teaching Assistants (TAs)**: Yujin Chen, Haoxuan Li, Chandan Yeshwanth
* **Student Tutors**: Shaotong Chen, Han Wu, Hongtai Yuan, Jinxin Ai, Estevao Gomes, Ignacio Dassori, Youssef Youssef

## Syllabus
* Lecture 1: Introduction to the lecture, Deep Learning, Machine Learning.
* Lecture 2: Machine Learning Basics, Linear regression, Maximum Likelihood
* Lecture 3: Introduction to Neural Networks, Computational Graphs
* Lecture 4: Optimization and Backpropagation
* Lecture 5: Scaling Optimization to large Data, Stochastic Gradient Descent
* Lecture 6: Training Neural Networks I
* Lecture 7: Training Neural Networks II
* Lecture 8: Training Neural Networks III
* Lecture 9: Introduction to CNNs
* Lecture 10: CNNs architectures 
* Lecture 11: Recurrent Neural Networks (RNNs) 
* Lecture 12: Transformers
